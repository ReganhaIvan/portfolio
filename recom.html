Movie Recommender System

Project Overview

The aim of this project is to understand the theory behind a recommendation system, and to work on the different methods that can be used for this type of problem. In this case, it's a recommendation system for movies. 2 different approaches were taken into account: Content based filtering and Collaborative filtering. 2 notebooks were built: one to study different methods and another to propose a final solution.

About the Dataset:

The dataset used in this project contains information about the movies and the user ratings.
